## Common Tasks

### Add Signature Area
1. Select Shape (rectangle)
2. Set color to light gray
3. Set stroke width to 2px
4. Add text "Signature: _______________"

### Mark Document Status
1. Select Watermark
2. Enter "DRAFT" or "APPROVED"
3. Set opacity to 0.4
4. Choose red or green color

### Add Company Logo
1. Select Image tool
2. Choose logo image
3. Position at top-right
4. Set size ~100px

### Highlight Important Text
1. Select Highlight tool
2. Choose yellow color
3. Set opacity to 0.3
4. Duplicate and position over text
